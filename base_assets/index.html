How to get started

When you launch the application, a new map is automatically created. You can start placing tiles on it by exploring the side menu containing the basic tiles automatically built into the software. To save your map, simply open the "map" menu located above the map and click on "save". You've created your first map! To go even further, you can create your own tiles that you can then place on your maps. To do this, simply open the "Tile mode" side tab, create your tile in the same way as you created your map, then save it by clicking on the "Tile" menu above the grid and then "Save". You can now return to the "Map mode" tab and use your own tile. Similarly, you can create your own blueprints by opening a new map, drawing your blueprint using tiles, then saving it under 'Blueprint' then 'Save'. They will then appear in the 'Blueprints' category of the 'Map mode' side tab.

Main features

  • Top menu:

    The top menu allows you to manipulate your data:
    • The "Project" tab lets you manage a project, which is a folder containing all your maps. When you open a new project or load an existing one, it will appear in the 'Project' tab on the side. In this tab, you can link maps to your project, which will appear in the tab. You can use "Delete" to completely delete a project.
    • The 'Map' tab lets you manage your maps. You can open a new blank map with 'New', load a saved map with 'Load', save a map with 'Save' and delete a saved map with 'Delete'.
    • The 'Tile' tab lets you manage your custom tiles. You can open a new blank grid and switch to "Tile mode" using "New", load a saved tile so that you can modify it using "Load", save a tile that you have created using "Save" and delete a saved tile using "Delete".
    • The "Blueprint" tab lets you manage your custom blueprints. You can open a new blank map with 'New', load a saved blueprint so that you can modify it with 'Load', save a blueprint that you have created with 'Save' so that it appears in the 'Blueprints' category of the 'Map mode' tab, and delete a saved blueprint with 'Delete'.
    • The "Edit" tab lets you undo an action with "Undo" or redo an action with "Undo".

  • Side menu:

    • The "Map mode" menu contains all the elements you can use to create a map, predefined tiles included in the software, user tiles and blueprints created by the user.
    • The "Tile mode" menu contains all the tools needed to create custom tiles, with the option of choosing a colour or creating your own using the RGB buttons. You can store up to 10 custom colours.
    • The "settings" menu contains the customisation options you need to use the software, such as the ability to delete, hide or display the grid on top of the tiles. You can also choose the scrolling speed in the "Map mode" menus.
    • The 'Project' menu displays the name of the current project if one is open, as well as the names of the maps inside.

Keyboard shortcuts

  • CTRL-z: Undo
  • CTRL-y: Redo
  • Arrows: Move the map
  • Right click (hold): Move the map
